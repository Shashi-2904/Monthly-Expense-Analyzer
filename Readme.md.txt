# 📊 Monthly Expense Analyzer – 2 KPI Dashboard

This Power BI project includes two KPIs to help visualize and manage monthly finances.

## ✅ KPIs
- **Total Income**
- **Total Expense**

## 📁 Files Included
- `Monthly Expense Analyzer.pbix`: Power BI dashboard
- `Monthly Expense Analyzer.csv`: Sample dataset
- `README.md`: Project details

## 🧠 Tools Used
- Microsoft Power BI Desktop
- DAX for KPI logic
- CSV for data input

## 🚀 How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Explore the KPI Cards and visuals.
3. Add your own data or build on top of the project.
